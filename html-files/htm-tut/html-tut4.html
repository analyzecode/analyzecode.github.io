Html Elements

An HTML file is made of elements.These elements are responsible for creating web pages and define content in that webpage.


An element in HTML usually consist of a start tag <name>, close tag </tag name> and content inserted between them. Technically, an element is a collection of start tag, attributes, end tag, content between them.
