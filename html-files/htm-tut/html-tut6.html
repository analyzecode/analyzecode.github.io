Html <p> tag

The <p> tag defines a paragraph. Browsers automatically add a single blank line before and after each <p> element.

For example :

<!DOCTYPE HTML>
<html>
<head>
<title>Basic Page Structure</title>
</head>
<body>
<h1>Use of <p> tag</h1>
<p>This is a Paragraph</p>
</body>
</html>
